# Dorah Portfolio Frontend (React + Vite + Tailwind)

## Run
1) Install:
```bash
npm install
```
2) Start:
```bash
npm run dev
```

## Backend
Set API base in `.env`:
```
VITE_API_BASE=http://localhost:5000
```

Endpoints:
- GET  /api/projects
- POST /api/auth/login
- POST /api/upload
- POST /api/projects
- DELETE /api/projects/:id
